
# Node.js HTTP POST Requests with Axios

## About

This project demonstrates how to send HTTP POST requests using Axios in Node.js.

## Features

- Send HTTP POST requests
- Add custom request headers
- Send JSON request body
- Receive server response

## Technologies Used

- Node.js
- Axios

## Installation

```bash
npm install axios
```

## Run

```bash
node index.js
```
